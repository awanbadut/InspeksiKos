# InspeksiKos 🏠

Platform audit properti berbasis Cloud dengan Gemini AI Vision + Rule-Based Algorithm.

## Tech Stack
- **Frontend**: Next.js 14 (TypeScript)
- **Backend**: NestJS (TypeScript)
- **Database**: PostgreSQL (Supabase)
- **AI**: Gemini 1.5 Flash Vision API
- **CI/CD**: GitHub Actions
- **Container**: Docker

## Quick Start
```bash
docker compose up
```
